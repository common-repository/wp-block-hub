== Description ==

**[WPBlockHub.com](http://www.wpblockhub.com)** is a network for sharing Gutenberg Block Templates, you can create block template from your website or localhost and share on wpblockhub.com, people will able to use your block template via **WP Block Hub** plugin. This plugin doesn't provide any blocks right now, you can use your favorite [blocks plugin](https://wordpress.org/plugins/tags/gutenberg-blocks/) available free at wordpress.org repository. you will need to add the block plugin list via meta box input fields you used to create block template, so people can install required block plugin to work on their site.

You will need to get an API key to submit gutenberg block templates on http://wpblockhub.com, you can get your [API key from here](https://wpblockhub.com/api-key/) its free and you will able to create API key after register on the site.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>WP Block Hub</strong>" activate it.<br />

After activate plugin you will see "WP Block Hub" menu at left side on WordPress dashboard. you can search to find your
blocks and download to import via reusable blocks.
